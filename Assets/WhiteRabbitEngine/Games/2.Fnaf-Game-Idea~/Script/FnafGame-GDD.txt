Fun Potential: The use of security cameras to track the animatronics' movements has high potential. It can create a sense of voyeurism and suspense. The player is not directly interacting with the danger, but observing it from afar, which can be very tense.
Challenges: It needs to be more than just watching screens. There needs to be a reason to monitor specific areas, and information gained from it needs to be useful.
How to make it more fun: Add different cameras with different perks. Some cameras can have better vision but less range, or viceversa.
Evasión (Evasion):
Fun Potential: Hiding, sneaking, and using distractions are core horror mechanics. Successfully outsmarting the animatronics will provide a strong sense of accomplishment.
Challenges: The mechanics need to feel responsive and fair. If the player feels like evasion is random or the animatronics can "cheat," it will be frustrating, not fun.

Fun Potential: This adds a layer of strategic resource management. Players must decide what to prioritize (lights, cameras, etc.), which can lead to tense situations and difficult choices.
